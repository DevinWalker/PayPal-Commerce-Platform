PayPal Commerce Platform - CP_O_R

Description

This example application corresponds to the PayPal Commerce Platform, and serves as a demonstration of the various products of the PayPal Commerce Platform. The name of this sample "CP_O_R" stands for the 3 products demonstrated here. "CP" stands for Connected Path Upfront Onboarding. Using Connected Path Upfront Onboarding follows the use-case where a Merchant would "setup" their Payment capabilities on the Partner's Platform, before accepting a Payment. The "O" stands for Orders via Checkout which uses v2 of the Orders API. And finally, the "R" stands for Refunds which uses v2 of Payments/Refund API.
